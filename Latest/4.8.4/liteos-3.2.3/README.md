# toolchain说明

适用于LiteOS 3.2.3， 对应海思arm-himix100-linux toolchain
fs version： 涂鸦配置保存在文件系统中的形式，需要开发者指定保存路径。
flash version： 涂鸦配置保存在flash分区中，需要开发者实现分区的划分和读写操作。

#arm-himix100-linux-gcc --version
